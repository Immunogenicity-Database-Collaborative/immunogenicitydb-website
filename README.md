# Immunogenicity Database Collaborative Website

Static GitHub Pages draft for `immunogenicitydb.org`.

## Files

- `index.html`: Home page
- `data.html`: Dataset downloads, publications, and version history
- `roadmap.html`: IDC database expansion current activities
- `community.html`: Contribution and contact information
- `assets/styles.css`: Shared styles
- `assets/main.js`: Mobile navigation and footer year
- `assets/idc-logo-header.png`: Optimized IDC logo used in the site header
- `assets/idc-banner-home.png`: Optimized IDC horizontal banner used on the home page
- `assets/idc-logo.png` and `assets/idc-banner.png`: Original supplied image files

## GitHub Pages

This site does not require a build step. Publish the folder contents from the repository root, or place them in a `docs/` folder and configure GitHub Pages to serve from `docs/`.

The dataset link points to the IDC-DB GitHub repository and the manuscript resource link points to the BioRxiv preprint. The documentation link remains pending until a stable data dictionary or methods resource URL is available.
